===================
Web Single App Logo
===================

This module allows to use a single logo in the views for all the companies.
It will not affect the company logo when we are printing or sending emails.
It is useful in order to have a logo for the App and another one for the
external documents.

Configuration
=============

* Access `Settings / General Settings`
* Upload a shared logo
