Signup User Account Activation Approval Process Odoo Apps

Sign up User Approval Odoo App is used to allow admin or manager to easily approve or reject new sign up users. Users can not directly login into account till it's not approved or verified by admin or manager. Users get email notification if the admin or manager approves or rejects the users account. Only approved users can log in into the website and restricted users can't log in.

Features

Admin can Approve/Reject Sign up Users

Admin or manager can approve or reject new sign up users.

User can't Directly Login into Account

If user till not approve by admin or manager, user can't directly login into account.

Approved User can Easily Login into Account

Approved user can easily login into account and display login screen.

Email Notification by Approval and rejection of Account

User gets email notification when approved account and reject or block account.
Demo Request Video Tutorial

Select 'Free sign up' Option on Customer Account

Under general settings, select 'Free sign up' option on customer account.

Sign up User Account

User has to fill sign up form in that write details of email, name & password after that click on 'Sign up' button.

User Account Registered Successfully!

When user click on the signup page, it will show a 'Registered Successfully!' message. after that, The administrator or manager gets the user's details and can approve or reject the user account.

If the user till not approved by the admin or manager and they tried to login then it will show 'Account Already Registered' message like below.

User Approvals

Goto Settings -> Users & Companies -> User Approvals in that list of sign up users.

Need Approval: In that admin or manager has to 'APPROVE' or 'REJECT' new sign up users.

APPROVE: After clicking on 'APPROVE' buton user gets email notification for approval and user can login into account.

Reject: After clicking on 'REJECT' buton user gets email notification for reject and user can not login into account.
