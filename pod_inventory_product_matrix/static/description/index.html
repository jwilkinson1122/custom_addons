Set Up

To begin, ensure that you have the Inventory Product Matrix downloaded and installed on your system.
To Configure this app, go to Odoo Dashboard >> Apps >> Search for "Inventory Product Matrix" and you should be able to see it.
After you activate it, you will be able to see the Variant grid entry option inside the Inventory setting which will be auto-enabled.
Do not disable it as it will uninstall this app.

Work Flow

Go to the Products page select any product and go to the Attributes & Variants section and add attributes. Once you add more then one attribute value you will see a new field below titled"Inventory Variant Selection"
Here you should be able to see three options.
1. Normal Entry
2. Order Grid Entry
3. Product Configurator
If you select Normal Entry, then you will be able to add Products in Inventory lines normally.
If you select Order Grid Entry, You will see a grid view open and you will be able to fill in details with it.
In the "Additional Info" tab, you will be able to see option "Print Variant Grids" if you check it, You will see a grid view of the products variants in the report of this picking.
If you select Product Configurator, You will see a matrix view open and you will be able to fill in details with it.
In the "Additional Info" tab, you will be able to see option "Print Variant Grids" if you check it, You will see a Matrix view of the products variants in the report of this picking.
