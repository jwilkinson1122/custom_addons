Odoo Amazon S3 Connector

Odoo is an Enterprise Resource Planning(ERP) system which includes CRM, Inventory, Warehouse etc. Amazon S3 or Amazon Simple Storage Service is a service offered by Amazon Web Services that provides object storage through a web service interface. Amazon S3 uses the same scalable storage infrastructure that Amazon.com uses to run its global e-commerce network.

  • Installation
  • How its work

Installation

  • Download odoo_amazon_s3_connector module and place into addons directory
  • Restart Odoo server / service
  • Open setting, activate Developer Mode
  • Go to Apps
  • Update Apps List from top menu
  • Search the Amazon S3 and install it
  • Happy to use Amazon S3


How Its Work


  • Install Amazon S3 Connector
  • Go to Amazon S3
  • Fill all required credential fields

  • Go to Contacts
  • During contact create/edit, attachments can sync with Amazon S3
  • Export file button sync attachments from ODOO to Amazon S3
  • Import files button sync attachments from Amazon S3 to ODOO

  • In Amazon S3, documents & files are sync successfully.
