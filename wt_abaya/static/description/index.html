Configuration

  • MEASURMENT CREATION

    • 'Measurment type' menu added inside pos configuration menu, There are several types of measurments are listed or you can create new one.

  • MEASURMENT IN POS PRODUCT CATEGORIES

    • 'Pos Product Categories' inside we have added new fields 'Measurment Type' and 'Measurment Unit', You just select pos product categories submenuitem and new page will be open which show product categories and measurment type created in measurment creation.

  • TAILOR PRODUCT SELECTION

    • Additional information like 'Is Tailor Product' are mentioned in the product view where this boolean is selected then only you can add a measurment.

  • MEASURMENT OPTION

    • We have added a new 'Measurment' button in the customer list screen, When you click on the 'Measurment' button it will open 'measurment Popup' where you can enter details.

  • SHOW MEASURMENT DETAILS

    • When you click on the "Show Measurment" button it will show you all listed measurments.

  • MEASURMENT PAGE

    • We also add a new one page called 'Measurment' where every detail is automatically added from pos.

  • SET MEASURMENT

    • We add a new button "Set Measurment" in the pos order screen, When you click on the button it will allow you to Set Measurment for a particular order line according to customers.

  • MEASURMENT AND UNIT

    • Also you can see measurments and measurments unit in the pos order backend side.
