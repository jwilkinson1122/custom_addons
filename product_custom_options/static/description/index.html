Odoo Product Custom Options

Set custom options for products in Odoo!!

Odoo Product Custom Options allows you to manage custom options without making product variants. All the options of the same product are shown on the same order line with their total price computed automatically.

Product Custom Options taking over Variants
With this module, you can create a custom option for products as per your requirement. Unlike variants, you do not need to manage the inventory of category custom options. Hence, simply select the options while creating sales orders.
The custom option shows the details of all the available options for you to choose from and add them to the cart with the details of the chosen option.
Create Custom Options for Product in Odoo
You can create common custom options for the product categories. No need to manage multiple stocks for custom options. Custom options apply to all the products which come under that category.
While creating custom options, you can choose from various options for creating custom options such as dropdown, checkbox, etc. Also, you can choose the field as required and set the default option.
