Features

POS Configuration for Create Sale Order

User can enable or disable create sale order option from POS configuration.

Create Quotation/Sales Order

User have option create sale quotation/order directly from POS.

Add Note for the Sale Order

User can add note for the sale order in the POS.

Sales Quotation/Order Filter By Create From POS

User can easily filter by record of created from POS.

Sales Quotation/Order Group By POS Session

User can easily group by record of POS session in sales quotation/order.

Customer and Product Validation

User can see warnings if he/she forgets to choose customer or forgets to select product.
Demo Request Video Tutorial

POS Configuration for Create Sale Order

Under POS configuration, User can enable/disable create sale order option to create sale order from POS.

Create Sales Order Button Appear in POS Screen

In POS Screen "Create Sales Order" button appear, User have to select customer, products and click on create sales order button to create sale order directly from POS.

Create Sales Order Popup

Once user click on create sales order button then open the popup to choose state Draft or Confirm of the sale order.

User can add note for the sale order.

Pop-up Message Display After Successfully Created Sales Order

User can view the popup message after successfully created sales order, When you click on the sale order number it opens quotation/sale order in the back-end.

Sale Orders Smart Button in POS Session

User can view "Sale Orders" smart button to open sales order created from those POS session.

Created Sales Order from POS

User can view the created sales order with products and POS session reference and also view the added note from POS.

Sales Quotations/Orders Filter By Create From POS

Sales Quotations/Orders Group By POS Session

Warning/Validation

If user tries to create sales order without selection of customer, Warning raised.

If user tries to create sales order without selection of product, Warning raised.
