Sales Product Attributes Size Matrix Odoo Apps

Product Size Matrix Odoo App helps users to display product variants sizing in matrix view for sale order. Once the module is installed product variants and attributes would be created for "Textile Sizing" or "Footwear Sizing" and applied to the product. Users can update product pricing according to their sizing and also update variant quantity from product matrix on sale order.

Features

Display Product Variant Sizing in Matrix

User can see product variant sizing like "Textile Sizing" and "Foot Ware Sizing" in matrix view.

Update Product Variants Price

User can update product variants price according to their size.

Increase/Decrease Product Quantity in Matrix Popup

User can easily increase or decrease product quantity in matrix popup on sale order.

Warning Raised While Tick Twice Sizing

When user tick on "Textile Sizing" and "Foot Ware Sizing" both for single product, Warning raised.
Demo Request Video Tutorial

User Configuration

Under user configuration, Tick on "Access Size Matrix On Product" checkbox.

Product Attributes

Goto Sales -> Configuration -> Settings in that user can see product attributes with their values.

Enable Product Sizing Checkbox

User can select any one of product sizing checkbox, either "Textile Sizing" or "Foot Ware Sizing" for the product.

When user tick on product sizing, Attribute is applied to the product,also there will be prices attached to different product variants which are set on the product variant page.

Add Product Matrix on Sale Order

User can create sale order with product matrix for that click on "Add Product Matrix" button.

Matrix Availability Popup

User can select product so their attributes is auto fill on popup.

User can enable product's attributes and also can update quantity then click on "ADD VARIANTS" button.

Added Product Matrix

User can see added product matrix or variants on sale order.

Warning/Validation

When user tick on "Textile Sizing" and "Foot Ware Sizing" both for single product, Warning raised.
