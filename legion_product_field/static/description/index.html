Product Fields

Byte Legion Private Limited

Overview

Odoo Application (Product Fields) will add the product fields for the Sale Order in the Products Details. Moreover, it will help to assign the licence number for the customer.

Feature

Addition of Product Fields for Products.
Addition of Licence Number for Customer.

How To Work

We just need to install this Module in our system, it will help us to assign the licence number to the customer.




Odoo Application (Product Fields) will add the product fields for the Sale Order in the Products Details. As we can see the added fields (Batch Number, Manufacturing Date, Expiry Date, Pack Size and MRP).




Now we can see that additional added product details in sale order and invoice order line. Also, we can see the customer's licence number in the form.
